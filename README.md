# 小旅的个人网站 (XiaoLv's Personal Website)

欢迎来到小旅的个人网站！这是一个采用极简安静风格（莫兰迪色系）设计的静态响应式网页。

## 🌟 项目特点
- **纯静态技术栈**：HTML5 + CSS3 + 原生 JavaScript，无需安装任何环境，双击即可运行。
- **设计风格**：低饱和度、低对比度的莫兰迪色系（浅灰蓝、米白、浅卡其），动效克制。
- **响应式布局**：完美适配 PC 端、平板和手机端。
- **零依赖**：使用 CDN 引入 Tailwind CSS 和 FontAwesome，无需 npm 安装。

## 📂 文件结构说明
- `index.html`：网站主页（包含首屏、人生轨迹、兴趣爱好、联系方式）。
- `css/style.css`：自定义样式文件，包含动画关键帧和部分特定样式。
- `js/main.js`：交互逻辑，包含滚动动画、复制功能、导航栏显示隐藏等。
- `pages/`：存放子页面模板（如经历详情页、爱好详情页）。
- `assets/images/`：**存放你自己的图片的地方**。

## 🚀 如何运行
1. **无需安装任何软件**：这是一个纯静态网页。
2. **直接打开**：在文件夹中找到 `index.html` 文件，**双击**它，即可在你的默认浏览器（如 Chrome, Edge, Safari）中打开并浏览。

## 🖼️ 如何添加你自己的图片（超级简单！）
你**不需要修改任何代码**，只需要把你准备好的图片放入 `assets/images` 文件夹，并**严格按照以下名称重命名（必须是 .jpg 格式）**，网站就会自动加载它们！

### 1. 替换首屏照片
- 将你的照片命名为：`home-photo.jpg`
- 放入 `assets/images` 文件夹中。
- 刷新网页，首屏左侧的照片就会自动变成你的照片。

### 2. 替换经历/爱好详情页图片
将对应的图片命名为以下名称（支持1-5张图片），放入 `assets/images` 文件夹中。当你点击对应卡片进入详情页时，图片会自动以画廊形式显示：

**图片命名规则**：`[拼音]-[数字].jpg` (例如：`xuzhou-1.jpg`, `xuzhou-2.jpg`)

**如何修改图片下方的文字描述？**
用文本编辑器（如记事本）打开 `pages/detail.html`，在大约第 100 行找到 `const imageDescriptions = { ... }`，按照里面的格式添加你自己的文字描述即可。如果没有配置，会自动显示“光影留念”。

*(注意：如果你还没有放入图片，网站会提示“暂无影像记录”，不会出现破图的尴尬情况。)*

### 3. 验证复制功能
在“联系方式”模块，点击微信或公众号文字，页面顶部会弹出“复制成功”的提示。你可以新建一个文本文档，按 `Ctrl+V` (或 `Cmd+V`) 粘贴，看看是否成功复制了对应的内容。

## 💡 给新手的进阶建议
- 如果你想进一步学习，可以尝试修改 `js/main.js` 里的动画速度。
- 可以在 `index.html` 的 `<script>` 标签中修改 Tailwind 的自定义颜色（如 `morandi-blue`），打造属于你自己的专属色系。
